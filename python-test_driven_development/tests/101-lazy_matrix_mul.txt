=================================
How to Use 101-lazy_matrix_mul.py
=================================

This module defines a matrix multiplication function ``lazy_matrix_mul(m_a, m_b)``
that multiplies two matrices using NumPy if available, with proper error handling.

Usage
=====

``lazy_matrix_mul(m_a, m_b)`` returns a new matrix representing the product
of ``m_a`` multiplied by ``m_b``.

Basic example:
::

    >>> lazy_matrix_mul = __import__('101-lazy_matrix_mul').lazy_matrix_mul
    >>> m_a = [
    ...     [1, 2],
    ...     [3, 4],
    ... ]
    >>> m_b = m_a
    >>> print(np.array(lazy_matrix_mul(m_a, m_b)))
    [[ 7 10]
     [15 22]]

Non-square multiplication:
::

    >>> m_a = [[1, 2]]
    >>> m_b = [
    ...     [3, 4],
    ...     [5, 6]
    ... ]
    >>> print(np.array(lazy_matrix_mul(m_a, m_b)))
    [[13 16]]

Floating-point matrices:
::

    >>> m_a = [
    ...     [1.2, 5.5, 6.2],
    ...     [4.66, 12.3, -9.2]
    ... ]
    >>> m_b = [
    ...     [5.0, 3.3],
    ...     [-2.9, 4.4],
    ...     [7.2, 4.4]
    ... ]
    >>> print(np.array(lazy_matrix_mul(m_a, m_b)))
    [[ 34.69   55.44 ]
     [-78.61   29.018]]

Error Handling
==============

Missing arguments:
::

    >>> lazy_matrix_mul(m_a)
    TypeError: lazy_matrix_mul() missing 1 required positional argument: 'm_b'

::

    >>> lazy_matrix_mul()
    TypeError: lazy_matrix_mul() missing 2 required positional arguments: 'm_a' and 'm_b'

Invalid shape:
::

    >>> lazy_matrix_mul([[1, 2], [3, 4]], [[1, 2], [2, 3], [4, 5]])
    ValueError: m_a and m_b can't be multiplied

Empty matrices:
::

    >>> lazy_matrix_mul([[]], [[5, 6], [7, 8]])
    ValueError: m_a can't be empty

::

    >>> lazy_matrix_mul([[5, 6], [7, 8]], [[]])
    ValueError: m_b can't be empty

Invalid data types:
::

    >>> lazy_matrix_mul("not a list", [[1, 2]])
    TypeError: m_a must be a list

::

    >>> lazy_matrix_mul([[1, 2]], "not a list")
    TypeError: m_b must be a list

Non-number elements:
::

    >>> lazy_matrix_mul([[1, "x"]], [[3, 4]])
    TypeError: m_a should contain only integers or floats

::

    >>> lazy_matrix_mul([[5, 6], [7, 8]], [[5, "6"], [7, 8]])
    TypeError: m_b should contain only integers or floats

Unequal row lengths:
::

    >>> m_a = [[1, 2], [3, 4, 5]]
    >>> m_b = [[1, 2], [3, 4]]
    >>> lazy_matrix_mul(m_a, m_b)
    TypeError: each row of m_a must should be of the same size

::

    >>> m_a = [[1, 2], [3, 4]]
    >>> m_b = [[1, 2, 3], [4, 5, 6]]
    >>> lazy_matrix_mul(m_a, m_b)
    TypeError: each row of m_b must should be of the same size
